# 主にプレイしたゲーム (Score : 83)
## シレン5+
DS版未プレイ。
シレン4もやってないので、夜とか技とかの4からの新システムに最初はかなり戸惑ったものの、慣れてくれば稼ぎに使えるしで、持ち込みダンジョンでならありかなと思った。
新システム使いこなせるかどうかで難易度が大きく変わる感じ。
Scoreはストーリークリアまで。

## CROSS BEATS　8月～　(Score : 80)
- 半年でRP1400までプレイ。総課金額は1000円強。
- 収録曲の傾向が独特。他スマホ音ゲーのデレステ、SB69と違い、ボーカル以外の曲もあり、棲み分けは上手く行っていると考えられる。
 - そんな収録曲は全音ゲーの中でもかなり好みであり、特に初期に解禁できる曲で気に入ったものが多い。
- AC版は数クレしかやってない。近くに置いてないので……。

### 運営
- イベントは累計スコアアタック（チーム戦）、ハイスコアアタック（個人戦）、ランダムセレクト祭、アンロックチャレンジ祭の繰り返しでマンネリ気味。
 - AC稼働後だからおざなりになっているのかな？
 - ACの新曲がiOSにあまり入らず、その逆も然り。仕方がないと言えば仕方がないか。

### UI
- 画面UIはかっこいいが、ユーザビリティには伸びしろがある。
 - 画面遷移にロードを挟みすぎてる。

### プレイ感
- 普通にプレイする分にはReflecやjubeatに通じる面白さがある。
 - Lv70近くの難曲も複数回プレイしていくうちに、だんだんスコアが上がっていくのは楽しい。
 - つまらない譜面がないわけではない。
 - 連続フリック、縦連、混フレは初見殺しで基本楽しくない。iPadでプレイすれば変わるかも。
 - それらを攻略する一番いい方法がネットでプレイ動画をみて、リズムを覚えること。
  - 他の音ゲーもそうだが、特にクロビは覚え要素が強いのでそれが言える。
  - ちなみに楽しくない譜面の殆どが何回プレイしてもピカグレが出ず、動画でリズムを覚えないと厳しいタイプの譜面。

### 好きな曲は
- Aqua light、 The Epic、 What Is Wrong With The World、 Othersideとやはりベテランの曲ばかりである。
 - やはり収録曲に偏りがないのが、デレステと比べてかなり良い。

## SHOW BY ROCK!!
キャラ超かわいい。アニメはここ数年見たものの中でも、かなり気に入る出来だった。
女の子がかわいいのもあるけど、男キャラもみんな魅力的だった。サンリオすげえな。

音ゲーも悪くはないのだが、３レーンしかないのであまり難しくないのと、ステータスでスコアが決まるシステム、故に、二ヶ月くらいで飽きてしまった。

曲は、バンドメインのゲーム、故に…、ボーカル曲ばっかりである。
好みが分かれる所だが、音ゲーでやるならインストの方が好みなので、クロビやBEMANI全般の方が好きなの多いかなと。

それでも好きな曲はあり、旅路宵酔ゐ夢花火なんかは全盛期のTEЯRAみたいな中毒性があり、しかも譜面もやりごたえのあるもので印象深い。
故に、CD早く買わなきゃね。

## デレステ (Score : 76.5)
課金額0。
### いいところ
- フルコンできるかできないかあたりの曲をプレイするのが楽しい
- フルコンやクリアで埋めていく楽しみがある（フルコンで石もらえるのもGOOD）
- 3Dの出来がとてもよい。鑑賞モードで楽しめるのもいい
- ストーリーがあり、キャラに愛着が湧く
- 無料でたくさん石を配ってくれる
- 上位譜面(Master)は曲によって個性が出ていて飽きない
 - これはかなりよい部分
- 機能追加がかなり早い
- 楽曲追加ペースも早く、解禁条件が基本的に楽で課金の必要性が（楽曲については）殆ど無い
 - アイドルはともかく、楽曲でランダムセレクト祭みたいなことをやらないのは非常に好感が持てる
  - ACの音ゲーだと当たり前のことだが
 - 特定のSSRアイドルを解禁しないとプレイできない楽曲……という要素も今のところない
- クロビよりスタミナがゆるい
 - レベルアップで上限が増える他、回復アイテムまで存在する
 - ましてやスタミナを使わないプラクティスモードまである

### わるいところ
- 他の音ゲーの比じゃないくらいラグる
 - アプデで改善したが、まだ他の音ゲーの水準に達していない
- ボーカル曲しかない
 - SB69同様仕方ないことだが
 - それでもバリエーションに乏しく、昔のナムコゲーの楽曲や元祖アイマスの曲のシンデレラガールズカバーみたいなのが入ってもいいかもしれない
- スコアが判定の良さ+パーティの強さ(+スキル運)で決まる
- スマホゲー特有のイベントのマンネリ感
- スライドの判定が運任せ
- Lv28以上の譜面につまらないものが多い
 - 特にスライドは判定がスッポ抜けることも相まって、ストレス要素になりがち
 - 個性的で印象に残るが、フルコン目指すとなると苦痛でしかない
- こずえちゃんはよ入れてくれ

## オーディンスフィア (Score : 95)
数年に1本レベルの名作。
音楽、ストーリー、グラフィックどれも最高峰。
特にお気に入りはタイタニア城下町。背景の街並みのワクワク感が半端ない。

### 不満点
- ダメージモーション中が無敵でなく、一部のボスで連続でドリブルされ死に、萎えることが多々あった
- 最終ストーリーとクリア後の要素があっさり終わってしまって残念

これらの欠点がなければ、生涯で最高のゲームだと思い込んでるダーククロニクル（97点）も超えうるソフトだった。

なお、PS2版も収録されているがチュートリアルをプレイした時点で、システムと操作感が微妙な出来であった。
そういう意味ではリメイク版は大成功であると言える。

## Splatoon (Score : 88)
### 一人用
ラチェクラみたいでかなり面白い。
マップは殆ど一本道で迷うことはなかった。分かりやすくていいが、ラチェクラみたいに広いマップを探索させる要素があっても良かったかもしれない。
難易度はそれほどでもなく、面数も少ないかったから、もう少しボリュームが欲しかったところ。

### オン
自分に合う武器を見つけられると途端に楽しくなる。96ガロンとかがお気に入り。
勝ち方がなってくるとやはり面白い。俺こんなに頑張ったのにあいつのせいで負けたわーとか思えたら、それはもう虜になっています。
あと１試合３分という

ステージは今でこそそこそこの数だが、初期はステージが少なく、また時限解禁の要素が多くて、すぐに飽きてしまいがちだった。
ただ、一緒にやる人がいると楽しい。
特にスカイプしながらはかなり盛り上がります。

### 総評
コレのためだけに、WiiU買ってもいいレベル。
逆に言うと、人に薦められる2015年度発売のソフトがこれしかない。

ゲームとは直接関係ないが、Amiibo買えたのが年明けてからだった。しっかりしてくれ任天堂。

## スーパーマリオメーカー (Score : 50)

今年一番のガッカリゲーム。
任天堂は2Dマリオに未練ありすぎでは。

作ったステージはせいぜい2個で、ほかの人のステージを遊ぶモードを中心にプレイ。

音符ブロックで曲作れたり、Amiiboで他のゲームのキャラになれたりできたのはよかった。
特に後者はクリア時、ミス時もBGMも凝っててよい（元の作品から引っ張ってきただけだが）。
できれば、他のキャラ特有の操作とかができるようになると良かったが、30種類くらいいたのでそれは流石に厳しいかな。

特定箇所に行くとクラッカー鳴ったり等、全くマリオ関係ない演出は入れた意味が分からない。

Splatoonといい時限解禁要素（こちらはディスクには最初から入ってて起動日数等で解禁される）が多いのが難点。
解禁システム自体はそこまで気にならなかったものの、要素を解禁しないとおける物がワンパターン過ぎてすぐに飽きてしまう。

また買った理由の一つが、エディットUIが気になるというものだったが、そこまで悪くはなかった（よくもない）。
むしろ気になったのが、他人のステージを検索するUIまわり。eshopといい任天堂は、ゲーム以外のUI部分が特に弱い。

結構厳し目の点数をつけたけど、マリオのステージ作りたい！という人なら全然遊べるソフトだと思います。
RPGツクールとかマイクラ好きな人ならいけるかな？


## Fallout3 (Score : 83)
オブリが面白かったから、そしてFO4が発表されたので、やってみた。
TESシリーズみたいな世界観が好きだから、合わないかなーと思っていたが意外といけた。
システムを理解できれば、面白くなる。

オブリと比較して良い点
- 戦闘は面白い
- 敵がキモくて印象に残る
- サバイバルしてるんだなぁって気分になれる
- ダンジョンが怖い。メインクエストで廃れたVaultの中を探索するのは（難易度も相まって）心折れそうだった。
- 主人公に幼なじみや家族がいて、ストーリーに人間味があって面白かった

オブリと比較して駄目な点
- 敵がキモくて生理的に無理な奴も（Gとか）
- 探索する楽しみはあるが、ぼろぼろな世界なのでただ歩いてるだけでは楽しくない（）
- 序盤でよく金欠、弾欠（？）になる
- 序盤きつい終盤ぬるいなゲームバランス

オブリ同様の不満点
- やはりフラグ管理が甘い
- やはりフリーズする

コーラ飲んで、ポテチ食いながらやるメリケン式ゲームプレイがおすすめ。

## Skyrim (Score : 85)
オブリビオンをクリアしたので、プレイ。

オブリと比較して良い点
- 戦闘が多少楽しくなった
- 料理が美味しそう
- 全体的にUIやシステムまわりが向上してる

オブリと比較して駄目な点
- UIは使い勝手こそよくなっているが、ボロい紙切れみたいなメニュー画面の方がよかった
- ドラゴンが空とんでるとこちらから攻撃する手段が減って、マンネリ、リオレウスが降りてくるのを待っている気分
- 設定上仕方ないが、雪降ってるエリアが多く、前作ほどのフィールドを歩く楽しみはなかった

そこまで前作とは変わってないので、オブリが好きならこちらも行けます。

## （参考）オブリビオン (Score : 91)
良い部分はかなり良い。中毒性がヤバイ。
今のところ、ベセスダのゲームで一番好きです。

ただ、不満点の多さのせいで、それが失われているのがもったいない感じ。
一応、気になった点を挙げておくが、世間で言われてることと大差無いです。
今からやるならSteam版をおすすめします。

評価点
- 歩いているだけで楽しい中性的な世界観と変化に富んだフィールド
- 音楽、グラフィックはただいいだけでなく、前述の世界観の良さをいい感じにひきたててくれる
- 万引きが楽しい
- サブクエストもメインクエストも面白い
- プレイスタイルをいくらでも変えられるので何周やっても意外と飽きない

不満点
- 戦闘は単調
- ダンジョンは中身がほとんど同じ
- フラグ管理が甘い
- ロードがやや長い(PS3版だからかな)
- 結構な頻度でフリーズする(PS3版だから)
